# Ellsworth American Bootstrap WordPress Theme

### Minimum Requirements
- Ruby 1.9.3p194
- Ruby Gems 1.8.24
- sass gem
- compass gem
- bootstrap-sass gem

### Placehold.it Tool  
* **Basic Usage:** http://placehold.it/widthxheight  
* **Color:** http://placehold.it/widthxheight/bghexvalue/texthexvalue
* **Text:** http://placehold.it/widthxheight&text=sample text

**Example:** *http://placehold.it/150x150&text=Hello World*

### Required Plugins for WordPress Installation

* Ambrosite Web Development Services: Next/Previous Post Link Plus for WordPress (http://www.ambrosite.com/plugins/next-previous-post-link-plus-for-wordpress)
* Really Simple Captcha
* Contact Form 7
* Random Image Plugin
* SlideShow Plugin by Sefan Boonstra
